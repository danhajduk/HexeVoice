# Setup

1. Create a virtual environment.
2. Install backend requirements.
3. Copy `scripts/stack.env.example` to `scripts/stack.env`.
4. Update backend and frontend commands if needed.
5. Start the backend with `PYTHONPATH=src .venv/bin/python -m hexevoice.main`.
6. Start the frontend from `frontend/` with `npm run dev -- --host 0.0.0.0 --port 8080`.
